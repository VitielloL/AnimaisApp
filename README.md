# AnimaisApp

CRUD e Filtros utilizando Angular + Node e MongoDB ou Json.
Hospedagem: https://animais-app.vercel.app/

--------------------------------------------------------------------------------------------------------------------------------------------

<i>Frontend:</i> 

npm i

npm install -g @angular/cli

ng add @angular/material

--------------------------------------------------------------------------------------------------------------------------------------------

Backend:

npm i express

npm i mongoose

--------------------------------------------------------------------------------------------------------------------------------------------

Json:

npm i json-server
